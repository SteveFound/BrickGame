# Brick Game

Brick breaker type game written in Godot 4.2 with GDScript. Hit the ball with
the paddle and destroy the wall !

## Controls:	
* A or Left arrow move left
* D or Right arrow move right.
* Game Controller - Left thumb moves paddle.
